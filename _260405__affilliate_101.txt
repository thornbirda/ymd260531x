[1. 프로젝트 개요]

본 프로젝트는 GitHub(지텁)의 무료 인프라를 활용하여
소상공인 대상 소규모 일대일 제휴마케팅 시스템을 구현하는 것을 목표로 한다.
전문 플랫폼(인포크링크 등) 없이도 핵심 기능을 무료로 구축할 수 있다.

핵심 목표:
  - 비용 제로(0원)로 제휴마케팅 시스템 구축
  - 스폰서별 일대일 전용 페이지 운영
  - 방문자 통계로 스폰서에게 성과 증명
  - 수기 정산으로 자동화 비용 절감


----------------------------------------------------------------
[2. 플랫폼 구조 및 도구]
----------------------------------------------------------------

기능                  | 도구                  | 비용
----------------------|-----------------------|------
웹페이지 호스팅       | GitHub Pages          | 무료
방문자/클릭 추적      | Google Analytics      | 무료
신청 폼 접수          | Google Forms          | 무료
DB / 정산 관리        | Google Sheets         | 무료
링크 단축 + 추적      | Bitly 무료플랜        | 무료
스폰서 소통           | 카카오톡 / 이메일     | 무료


----------------------------------------------------------------
[3. 핵심 전략: 루트별 일대일 대응]
----------------------------------------------------------------

복잡한 추적 시스템 대신, 스폰서마다 전용 페이지를 개별 생성하여 단순하게 관리한다.

폴더 구조 예시:
  thornbirda.github.io/sponsor/미장원A/  →  스폰서A 전용 페이지
  thornbirda.github.io/sponsor/미장원B/  →  스폰서B 전용 페이지
  thornbirda.github.io/sponsor/미장원C/  →  스폰서C 전용 페이지

각 페이지는 독립적으로 방문자 통계가 집계되므로
스폰서별 성과 측정이 가능하다.


----------------------------------------------------------------
[4. 운영 흐름]
----------------------------------------------------------------

단계별 프로세스:
  1. 소상공인 스폰서 섭외 (카톡/직접 방문)
  2. GitHub Pages에 전용 페이지 제작
  3. Google Analytics 연결 (클릭/방문 측정)
  4. 월말 통계 캡처 → 카톡으로 성과 보고
  5. 계좌이체로 수기 정산


----------------------------------------------------------------
[5. 모집페이지 구성]
----------------------------------------------------------------

GitHub Pages로 스폰서 모집 랜딩페이지를 별도 제작한다.
신청은 Google Forms로 연결하여 자동으로 Sheets에 저장된다.

모집페이지 필수 내용:
  - 운영자 서비스 소개 및 방문자 규모
  - 스폰서 비용 및 혜택 안내
  - 성과 보고 방식 설명
  - 신청 버튼 → Google Forms 연결
